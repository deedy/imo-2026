## imo-2026-04

- Distinct openings (from the retrieval lens):
  1. **"Forced-threat" reduction to a deterministic chip-transfer game.** Working out the cut mechanics explicitly: cutting from a point P on side BC to vertex A splits angle A into (α1,α2) with α1+α2=A, and produces two children triangles {B, α1, C+α2} and {C, α2, B+α1} (both sum to 180, since B+α1+C+α2 = A+B+C = 180). If Mulan sets α1 = θ exactly, child 1 already contains angle θ (instant win if Shan-Yu keeps it), so a rational Shan-Yu is *forced* to keep child 2 = {C, α2, B+α1} = {C, A-θ, B+θ}. This is not adversarial at all — Mulan fully controls the transition whenever some current angle A > θ: she may choose to subtract θ from that angle and add θ to *either* of the other two (her choice), leaving the third fixed. Iterating this "attack-the-current-max, transfer θ to a chosen coordinate" move only requires that some angle stay > θ, which is guaranteed at the very first step (a triangle's max angle is always ≥ 60°, so if θ ≤ 60° an attackable angle always exists at the start — verify max ≥ θ is preserved at each step too, since after the move max(new state) can still be shown ≥ 60° by the same identity, so the move is repeatable). This is the natural "opening" line to explore first.
  2. **A structural obstruction to opening 1 (found by direct computation, not conjecture): a residue-mod-θ invariant.** Under the forced-threat move family alone, every coordinate only ever changes by an integer multiple of θ (± θ per touch), so each coordinate's value mod θ is fixed once it is first set, all the way from the initial triangle. Hitting exactly θ (≡ 0 mod θ) requires some *initial* angle A_i^0 to already be ≡ 0 (mod θ), i.e. A_i^0/θ ∈ ℤ. Shan-Yu, choosing the initial triangle, can trivially avoid this (pick three generic angles summing to 180 with none an integer multiple of θ). **This proves the naive "always attack the max by exactly θ" strategy alone can never force a win, for any θ.** This is a genuine (proved, not conjectured) dead-end to flag for the outliner — Mulan must at some point use a *non-forcing* move (a split ratio not equal to θ), which hands Shan-Yu a real binary choice between {B, α1, C+α2} and {C, α2, B+α1}. The real content of the problem is in analyzing that genuine game tree — likely via a double-threat / fork argument (both children simultaneously "good" for Mulan) or a more refined invariant that isn't preserved mod θ.
  3. **Rational-relation / commensurability framing.** Reframe "θ is achievable" via an algebraic condition on θ relative to 180°: is 180/θ rational? An integer? The genuinely analogous crux (aimo-0355, see below) shows how a similar triangle-angle rational-linear-relation collapses to a cosine identity `cos(rα) = ±cos(sγ)` with an integrality obstruction on 2cos(θ). This suggests the final characterization of the winning θ is likely to be expressed via commensurability of θ with 180° (e.g. θ = 180°·p/q in lowest terms with some parity/coprimality condition on p, q — possibly q odd, or θ ≤ 60° combined with a rationality condition) rather than a clean inequality alone. Worth checking small explicit θ (60°, 90°, 45°, 36°, and an irrational θ) by hand/simulation to see which admit a finite forcing tree.
  4. **Symmetric double-threat / fork construction (borrowed technique, not yet built into a proof).** In many of the analogous adversarial games in the corpus (aimo-0445's "double threat/fork", aimo-0917's branch-splitting invariant), the winning player creates a position where *both* of the adversary's options are equally bad. Here that would mean choosing a split (α1, α2) — not necessarily α1 = θ — such that *both* children {B, α1, C+α2} and {C, α2, B+α1} are already forced-winning for Mulan (e.g., each already has an angle θ, or each is a position from which the forcing chain of opening 1 is guaranteed to terminate). This merges openings 1 and 2: use ordinary (non-θ) splits to walk the state into a region where *every* subsequent forced-chain (regardless of Shan-Yu's past choices) provably terminates, rather than trying to force termination via a single deterministic path.

- Candidate technique(s): forced-move / fork ("double threat") construction in adversarial games; invariant/monovariant analysis of a value mod θ (an *obstruction*, proved above, not just a heuristic); commensurability / rational-cosine-relation analysis (à la aimo-0355) to pin down the exact characterization; game-tree finiteness arguments (König's lemma / finite depth strategy trees) for the "for all Shan-Yu play, in finitely many steps" quantifier structure.

- Cheap-kill candidates:
  - The mod-θ obstruction above is a genuine one-line kill of any "just subtract θ from the max forever" approach — flag it so the outliner doesn't waste a round rediscovering it.
  - Sanity check: max angle of any triangle is ≥ 60°, so θ > 60° rules out the naive opening move at the very first step for degenerate/near-equilateral starting triangles chosen by Shan-Yu (he can start with all angles < θ if θ > 60°, e.g. all three angles between 55° and 65° won't work for θ near 60 exactly but for θ close to 180 he can certainly start with all angles small) — the "attack-the-max" style approach fundamentally cannot even begin unless the current triangle already has an angle ≥ θ. This suggests θ > 90° (where the initial triangle Shan-Yu picks could have all angles < θ trivially, e.g. equilateral 60/60/60 if θ>60) needs Mulan to actively *build up* an angle toward θ using non-forcing splits before any forcing move is available — a genuinely different regime from θ ≤ 60°. Good structural split point for casework: θ ≤ 60° vs θ > 60° (or vs θ ≥ 90°, etc.) is a natural first bifurcation, not yet a proof.

- Analogous past problems (cruxes):
  - **aimo-0917 (IMO Shortlist 2020 C8, "A and B blackboard game")** — genuinely analogous *mechanically*: player A picks two board numbers, player B chooses one of exactly two replacement values (x+y or |x−y|), mirroring Mulan proposing a split and Shan-Yu picking one of two resulting triangles. The load-bearing crux there — "split an invariant count N = N₊ + N₋ over the two possible adversary responses, and note that if N is not divisible by 2^{s+1}, it can't divide both summands, so at least one branch keeps the invariant" — is a strong template for how to force a favorable branch when Shan-Yu has a genuine (non-forced) choice, i.e., directly useful for openings 2/4 above where a real 50/50 choice must be analyzed rather than eliminated by an instant threat.
  - **aimo-0355 (USA TSTST 2020, "quirky triangles")** — analogous *domain* (a triangle's three angles under an integer-linear-relation / commensurability question) even though the game mechanics differ entirely. Its crux move — eliminate one angle via the angle-sum, collapse an integer relation `r1θ1+r2θ2+r3θ3=0` to `cos(rα)=±cos(sγ)`, then use that a monic integer polynomial evaluated at a rational number has denominator exactly q^k to force strong constraints (both denominators must be powers of 2) — is the right *style* of argument for pinning down exactly which θ (as a function of 180°) permit or forbid the phenomena at stake, useful if the final answer is expressed via rationality/commensurability of θ with 180° rather than a pure inequality.
  - **aimo-0445 (USAMO 2014 hexagon game)** — the "fork/double threat" crux (make a move whose one legal single-step adversary reply cannot block both of two independent winning completions) is the generic pattern to look for in opening 4.

- Dead ends (do not retry): the pure "always attack the current maximum angle by exactly θ, transfer to a chosen other angle" strategy, used alone and forever — **proved above** (not just suspected) to fail for every θ, because it induces a mod-θ residue invariant on each never-yet-attacked coordinate that Shan-Yu can set up-front to avoid ever equaling θ. Any approach that stops at "Mulan attacks the max by θ each round" is incomplete and will hit this wall; it must be supplemented by genuine non-forcing splits (real Shan-Yu choices) analyzed via a fork/invariant argument.
