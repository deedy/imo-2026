# Warning: static maximal-intersecting families can be infinite

On formal vertices $1,2,3,\ldots,z_2,z_3,\ldots$, define
\[
E_n=\{1,2,\ldots,n-1,z_n\},\qquad
C_n=\{z_2,z_3,\ldots,z_n,n\}\quad(n\ge2).
\]
The family $\{E_n,C_n:n\ge2\}$ is pairwise intersecting: same-letter sets share $1$ or $z_2$, and $E_n\cap C_k$ contains $z_n$ if $n\le k$ and contains $k$ if $n>k$.

It is in fact a plausible (and readily checkable) infinite self-blocking clutter: a finite transversal containing $1$ has a minimal subtransversal of type $E_n$, while one not containing $1$ has one of type $C_n$. Thus a static theorem asserting that every maximal intersecting up-family of finite sets with a finite member is finite-variable would be false.

The arithmetic greedy ordering must therefore be used. If $\mu(X)$ is the least integer at least $a_1$ with exact prime support $X$, then every rejected support $X$ has a disjoint accepted minimal support $H$ satisfying $\mu(H)<\mu(X)$. This weighted orientation is absent in the abstract example and should rule out an infinite staircase by multiplicative inequalities.
