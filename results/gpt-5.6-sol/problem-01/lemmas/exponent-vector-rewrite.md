# Exponent-vector rewrite lemmas

Let `x_1,\ldots,x_N` be nonnegative integer vectors over a finite coordinate set. A move selects two nonzero vectors `x,y` and replaces them by `x\wedge y` and `|x-y|`, coordinatewise.

## Coordinate-generated subgroup invariance

For every coordinate, the subgroup of `\mathbb Z` generated by all entries in that coordinate is invariant. For selected coordinate values `r,s`, the outputs are `a=\min(r,s)` and `b=|r-s|`. If `r\ge s`, then `(a,b)=(s,r-s)`, and each pair consists of integer linear combinations of the other because `r=a+b` and `s=a`; the case `s\ge r` is symmetric. Adding the unchanged coordinate entries preserves equality of the full generated subgroups.

## Strong normalization

The nonnegative integer
\[
\Phi=\sum_i\|x_i\|_1+\#\{i:x_i\ne0\}
\]
strictly decreases in every move. In each coordinate, `r+s` changes to
\[
\min(r,s)+|r-s|=\max(r,s),
\]
so the loss in the norm sum is `\sum_p\min(x_p,y_p)`. If the supports overlap, this loss is positive and the number of nonzero vectors does not increase. If the supports are disjoint, the norm sum is unchanged, while the outputs are `0,x+y`, so the number of nonzero vectors decreases by one. Thus `\Phi` strictly decreases in either exhaustive case.
