# Math Explorer Per-Role Rules

ALWAYS: Check whether the "two elements summing to 180" condition holds as the key invariant test before declaring a direction (because this distinguishes winning from losing θ in the triangle game structure, round 1).

NEVER: Trust that "all rational theta in (0,90)" work — theta=72 is rational and < 90 but Shan-Yu wins; the precise condition is 180/theta ∈ Z (round 1).

ALWAYS: For game problems involving triangle angles, immediately check the "bisection forces A/2 into both sub-tris" lemma and the "vertex = 2*theta forces theta into both sub-tris" instant-win condition (round 1).

ALWAYS: When the answer set appears to be {180/n : n ≥ 2}, check whether this is equivalent to some cleaner number-theoretic condition (like 180/θ ∈ Z_+) and verify with non-obvious cases like theta=180/7 (round 1).
