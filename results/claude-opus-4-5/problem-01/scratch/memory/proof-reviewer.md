# Proof Reviewer Memory

## Round 1 rules learned

ALWAYS: Verify the load-bearing mathematical identity independently before approving (re-derive from scratch, test computationally)
ALWAYS: Check that gcd conventions are stated when 0 is involved (gcd(a,0) = a is standard but should be explicit)
ALWAYS: Certify promotable lemmas only if they are self-contained, with statement and complete proof
NEVER: Trust a "clearly follows" or "by symmetry" without demanding the explicit step
